# 🏟️ Arena HUB 

O arena HUB é uma solução tecnológica desenvolvida para a disciplina de Programação Orientada a Objetos (POO) na CESAR School. O projeto visa solucionar a subutilização da Arena Pernambuco, conectando cidadãos, organizadores e administradores em uma plataforma web integrada para facilitar a divulgação de eventos e a gestão do espaço.

## 🚀 Tecnologias Utilizadas

- Java 17
- Spring Boot

## ✍🏻 Entregas do Projeto

### Entrega 01

- Histórias de usuário
- Protótipo Lo-fi no Figma
- Screencast do protótipo

### Entrega 02

- Implementar ao menos 2 histórias
- Issue/bug tracker
- Screencast

### Entrega 03

- Implementação de 2 histórias 
- Atualização do Issue/bug tracker
- Novo screencast

### Entrega 04

- Implementação de no mínimo 2 histórias
- Atualização do Issue/bug tracker
- Novo screencast
- Documentação para configuração de ambiente
